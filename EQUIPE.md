##  Metodologia de Trabalho

### Ferramentas Utilizadas
- **Controle de Versão:** Git + GitHub
- **Editor:** Visual Studio Code
- **Terminal:** PowerShell
- **Deploy:** Netlify (CI/CD)
- **Hospedagem Código:** GitHub
- **Database:** Firebase/Firestore + localStorage

### Fluxo de Trabalho
1. **Planejamento:** Definição de requisitos e arquitetura
2. **Desenvolvimento:** Implementação em branches/features
3. **Testes:** Validação individual e em grupo
4. **Revisão:** Code review entre membros
5. **Deploy:** Push para GitHub → Deploy automático Netlify
6. **Documentação:** Atualização contínua

### Práticas Aplicadas
-  Git Flow simplificado
-  Commits semânticos com emojis
-  Code review obrigatório
-  Testes antes de merge
-  Documentação contínua
-  Deploy automático
